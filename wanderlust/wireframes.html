Technical Specifications

Layout Constraints

  • Maximum Width: 1200px (within 960px-1320px requirement)
  • Container Padding: 20px on all sides
  • Grid System: CSS Grid and Flexbox
  • Breakpoints: 768px (tablet), 480px (mobile)
  • Font Stack: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif

Interactive Features

  • Image Slider: Auto-advance every 5 seconds, manual navigation
  • Gallery Lightbox: Click thumbnails to view full-size images
  • Form Validation: Real-time JavaScript validation
  • Mobile Navigation: Hamburger menu for mobile devices
  • Smooth Scrolling: CSS scroll-behavior and JavaScript

Accessibility Features

  • Alt Text: All images have descriptive alt attributes
  • Keyboard Navigation: Tab order and focus indicators
  • Semantic HTML: Proper heading hierarchy and landmarks
  • Color Contrast: WCAG AA compliant contrast ratios
  • Screen Reader Support: ARIA labels where needed
